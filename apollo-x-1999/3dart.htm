This is a tutorial for all you people that wants to learn
more things to do with your 3D program. These tutorials
are made specifically for 3D Studio MAX 2+ but the
techniques are pretty much the same for other 3D
programs out there. Before you start on these you should
be familiar with the controls and effects of the buttons
in 3D Studio MAX. I have zipped all the files, because you
probably want to download them and try for yourself
offline. So you don't have to use money while working to
be good at 3D. Remember, as for other art, that your
imagination is all that stops you. Try to use these
techniques to make other cool things, and vary your
stuff. Practice makes MASTER !!!!

- Tutorials For 3D Studio MAX -
