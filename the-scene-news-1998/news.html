Friday 21th August (8:00pm GMT)

You must really wonder about Divine sometimes. Today they release NFL GameDay '99 (c) 989 Studios which was way over the disk limit. They claimed that it was impossible to make it any smaller without ripping out all the sfx. Then Class comes along a little while after and discovers there is a massive Play-by-Play commentary file that can be removed thus reducing the total release size down to only 34 disks.

In a bid to go more main stream and expand Game Over Online (the guys who review warez releases) have removed all references to the pirate scene on all their pages. This means no more online NFO's or release listings which contain the group's name. Is this the start of Game Over turning commerical, ignoring the scene in a bid to make a profit like Acid and Ice did in the early nineties?

Vindicator from the Phrozen Crew sent me a letter saying that he used to run www.warezregulator.com aka Warez News, but after a few months rest he has returned, but changed the url to www.twofast.com

Wednesday 19th August (10:00pm GMT)

Congratulations to CLASS who celebrate their 300th release today with BASS Masters Classic: Tournament Edition. Contained in this release is a large NFO file with some words from Class to commemorate this achievement and a list of their 300 releases to date.

Razor 1911 have finally got their webpage back online at www.razor1911.com. Unfortunately nothing has been updated on it since it went down last year.

Trite's pheared.com is down till tommorow though I bet it could take it a little longer to return.

Scenelink are claiming that Legends Never Die (who have seemed to have done just that) have merged into Siege.

Friday 14th August (7:00pm GMT)

Trite's pheared.com has returned with all it's pages including Black Acid's infamous T.C.A.

Could Class be unofficially restarting the trend of releasing documents? In the old days most groups retyped the manuals, but no one could be bothered doing that these days. But scanning them and converting them to a Adobe PDF format is a viable option and that’s what Class has done with Spearhead_Dox-CLS.

Saturday 1st August (4:00pm GMT)

Razor and Paradigm have done it with golf games, now Paradigm is doing it to a racing game. What am I talking about? Releasing a game crippled and then releasing the missing levels (in this case tracks) as an additional addon in a bid to work around the disk limit. Why bother with a disk limit at all?
