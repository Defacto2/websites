Juice/Psychic Link is in my opinion the best demo of all times...

#Juice#

Juice,made by Psychic Link,is truly an amazing demo.Play this demo on a machine
that's well equipped,turn up the volume,watch the demo,listen to the music and just
enjoy the great work by great sceners. Read more about Psychic Link in the       
"The Best Demo Groups" section.

#Second Reality#

Second Reality,done by Future Crew,will stun YOU just as much as it did towards the
Assembly '93 attenders.This is a great demo,and is a close run with Juice.Future
Crew is an amazing group,nobody can say against that.Be sure to read about FC too.

#Stash#

Done by The Black Lotus.If you are negative towards windows 95/98 programming and so
on,check this one out.They have managed to pack amazing sound and graphix into a little
file,and it's truly a stunning demo.Im a DOS'er myself,but this one removes all doubt,
demos CAN be coded in Windows 95/98.

#Paper#

Done by Statix/Psychic Link & Vic/Acme.Full of cool ideas and creations.I was amazed of
how cool it was,and thus so small.I couldn't listen to the music,because I have SBawe64
but it can only pull it upwards.
