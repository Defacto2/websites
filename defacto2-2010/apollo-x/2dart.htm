This is a small tutorial on how to draw some cool stuff,using just your hand,
the mouse,and a drawing program.It's called pixel drawing,that is because you
draw the picture pixel by pixel,and use as few effects and tools as possible,
use your imagination to make things look cool.

The tutorial is done by me (biXen),and there will be new tutorials added to this
page frequently,so please check back to look for more tutorials.I will also get
other artists to add some tricks here,so hang in there.

Remember...Your best tool is your imagination...and DON'T give up,
hard practice gives results,always...I know it !!!


3D Text W/Light Reflection (by biXen)

First draw a letter using the line tool,with the width you want.Use antialias to
get the lines smooth looking.Then fill the inner hollowzone
with the same color as the lines.

Then it's time to add a little 3D effect.Draw a thin line on the side you want
to be visible,for the 3D effect,f.x. the right and under sides.Then clean draw it
till it looks nice.Maybe put some blur on places that are to visible.

Now lets add a little light effect onto the letter.Use your brush-like tool and
set it to fade to the background color (which you set to white).Then set it
to about 5 steps,and draw from the top corner and down-right,or wherever you
want the effect to be,often it looks better with two effects,it gives
the letter a plastic look which is quite oldskoolish :)


Grey Ball W/Coloured Fire Effect (by biXen)

First,make a grey ball,maybe with the brush tool,to make it seem more round.
Make it smooth in the edges,so it doesnt look like just a circle.

Then add some reflections to the ball with your brush tool.Use the same method
as with the text,fade 5 steps to white,and draw some lines,so that it looks
more round and like a ball,iron ball that is.

Then it's time to make the coloured fire effect.Start with a strong color,and
use the fade (30-50) steps to background (black) or transparent.Then draw the
lines from the balls different parts and to the right or left or up or down,
wherever you want the ball to look like it's going.Easy :)
