Scooby Snack Mag by Scooby Snack Magazine (1997)

This magazine was the first cracking magazine to be released for the Windows 95 operating system. Unfortunately the code of this feels very loose and unstable though there are some interesting features included. The magazine itself never gained much respect and thus never gained many interesting interviews or articles.
